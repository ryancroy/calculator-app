# Calculator App

A simple, beautiful web-based calculator with a premium glassmorphism design. Vibe coded in Google Antigravity with Gemini 3.

## Features

- **Core Arithmetic**: Add, subtract, multiply, divide, and calculate remainders.
- **Scientific Notation**: Automatically formats results larger than 9 digits.
- **Glassmorphism Design**: Modern, translucent UI with a vibrant background.
- **Responsive**: Optimized for desktop, tablet, and mobile devices.

## Technologies

- **HTML5**
- **CSS3** (Flexbox, Grid, CSS Variables, Backdrop Filter)
- **JavaScript** (ES6+)

## How to Run

1.  Clone the repository.
2.  Open `index.html` in your web browser.
